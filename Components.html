Components and Distribution

Components

We have divided our project into following components :

  • Faculty:
    1. Mark Attendance
    2. View list of students registered for a course, each student's attendance and details
    3. Faculty can also view his/her identity card if generated.

  • Admin:
    1. Harrithha : ADD STUDENT : A form for filling all data into the database and create initial login and password.
    2. Arvind : VIEW STUDENT DETAILS : All student names with their details displayed. Admin can click on any student name to view all details of that particular student.Add filters like : students who have selected particular course, attendance above 80%(grade accordingly) , Only female/male candidates, view(not edit) date wise attendance for each course, etc.
    3. Harrithha : EDIT DETAILS : Enter name of student to edit. Then display form containing all details of that student retrieved from database into textfields. Admin can edit whichever field. And click SAVE button to store updated info.
    4. Chidaksh : ID Card generation

  • Student:
    1. The student logs in and checks his/her attendance statistics for different courses/faculties.
    2. The student also has the option to register for course available and once he registers for the course, his name will appear for attendance whenever a faculty logs in to record the attendance.
    3. View their identity cards if generated.
